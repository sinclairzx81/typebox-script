## Overview

Enables TypeScript syntax highlighting inside `Type.Script(...)` calls, as well as common aliases such as `t.Script(...)`, `Script(...)`, `type(...)`, `t(...)` and `s(...)`.

Licence MIT

## Install

Search for `TypeBox Script` in the VS Code Extensions panel, or install via the command line:

```bash
code --install-extension sinclairzx81.typebox-script
```
